= pato

Command-line tool to manipulate the PATH variable.

== Usage

----
$ PATH=$( pato list )
$ PATH=$( pato push $PWD )
$ PATH=$( pato pushf $PWD )
$ PATH=$( pato insert 2 $PWD )
$ PATH=$( pato remove 3 )
$ PATH=$( pato swap 2 6 )
----
